### Start

To start, first we need to install the node modules with:

```js
npm install
```

### Run

To run we can run with developer mode and production mode with

```js
npm run build:dev
```

```js
npm run build
```

If we want to run the webpack server (the project run in port: 1234)

```js
npm start
```
